# Customer Behaviour and Product Performance Analysis

## Project overview

This project analyses the Olist Brazilian e-commerce dataset to explore customer behaviour, product performance, delivery experience and regional differences.

## Business questions

1. Which product categories generate the highest revenue and order volume?
2. Which product categories receive the highest and lowest customer ratings?
3. How do delivery delays affect customer review scores?
4. What is the repeat purchase rate?
5. Which regions have higher customer value?

## Dataset

Source: Olist Brazilian E-commerce Public Dataset on Kaggle

Main files used:

- Customers
- Orders
- Order items
- Products
- Order reviews
- Product category translation

Raw data files are not included in this repository. Download them separately and place them in `data/raw`.

## Planned workflow

1. Explore and assess data quality
2. Clean and transform the data with Python
3. Load the cleaned data into a SQL database
4. Answer the five business questions with SQL
5. Build an interactive Power BI dashboard
6. Translate findings into business recommendations
